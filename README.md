# Iron Pulse — Class & Coach

A static landing page for a gym training class: schedule, coach info, and contact. Built as a single HTML file with embedded CSS (Google Fonts, hero imagery from Unsplash).

## Run locally

**Option A — open the file**  
Double-click `index.html` or open it in your browser (File → Open).

**Option B — local server** (avoids any browser restrictions on `file://` and matches deployment):

```bash
npx --yes serve .
```

Then visit the URL shown in the terminal (often `http://localhost:3000`).

**Option C — Python** (if installed):

```bash
python -m http.server 8080
```

Open `http://localhost:8080`.
